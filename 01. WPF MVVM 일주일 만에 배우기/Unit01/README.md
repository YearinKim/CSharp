# 01장. 소개

## 이 책에 없는 것

* MVVM 패턴을 사용
* WPF 애플리케이션 코딩
* 필요할 때 더 많은 것을 스스로 찾을 수 있는 도구를 신속하게 제공

<br/>

## 전제 조건
* C#으로 애플리케이션을 생성해본 기본 경험이 있어야 한다
* 비주얼 스튜디오에 대한 실무 지식이 있어야 한다
* XML 구문에 대한 기본 지식을 갖추고 있어야 한다
* SQL 서버에 대한 기본 지식을 갖추고 있어야 한다
